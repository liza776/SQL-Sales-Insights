# SQL-Sales-Insights
SQL-based analysis of pizza sales data to identify revenue trends, peak hours, and top-performing products using MySQL.

# 🍕 Pizza Sales Analysis

### Project Overview
This project involves a comprehensive analysis of a fictional pizza shop's sales data using MySQL. The goal was to extract actionable insights regarding customer behavior, product popularity, and revenue trends.

### Key Features
* **Basic:** Total orders, total revenue, and highest-priced pizza.
* **Intermediate:** Peak hour distribution and average pizzas per day.
* **Advanced:** Cumulative revenue tracking and category-wise rankings using Window Functions.

### Tech Stack
* **Language:** SQL (MySQL)
* **Concepts:** Joins, Subqueries, Window Functions (RANK), and Aggregations.
